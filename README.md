# selenium-add-to-basket-tests

Учебный проект с автотестами на **Python + Selenium**.
Тесты проверяют работу интернет-магазина при **добавлении товара в корзину**.

Проект выполнен в рамках курса **«Автоматизация тестирования с помощью Selenium и Python» (Stepik)**.

---

## Что проверяют тесты

В проекте реализованы автотесты для страницы товара интернет-магазина:

* добавление товара в корзину
* проверка названия добавленного товара
* проверка стоимости корзины
* проверка нескольких promo-акций с помощью параметризации PyTest
* поиск бага на одной из promo-страниц
* отрицательные проверки (отсутствие success-сообщения)

Тесты написаны с использованием паттерна **Page Object**.

## Установка зависимостей

Создать виртуальное окружение:

```
python -m venv venv
```

Активировать виртуальное окружение.

Windows:

```
venv\Scripts\activate
```

Linux / macOS:

```
source venv/bin/activate
```

Установить необходимые библиотеки:

```
pip install selenium pytest
```

---

## Запуск тестов

Запуск выполняется через PyTest:

```
pytest -s -v test_product_page.py
```

Параметр `-s` нужен для вывода проверочного кода из alert (используется в заданиях Stepik).
Параметр `-v` включает подробный вывод результатов тестирования.
